# PluginHelper 

This project is a Kotlin DSL that helps you develop Minecraft plugins.  
Complex tasks can be easily solved. 

---
### groovy 
```groovy
repositories {
    mavenCentral()
}

dependencies {
    compileOnly "org.pluginhelper:PluginHelper-core:1.0.1"
}
```

### kotlin
```kotlin
repositories {
    mavenCentral()
}

dependencies {
    compileOnly("org.pluginhelper:PluginHelper-core:1.0.1")
}
```
